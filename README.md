# raymo-et-al-2026-divorce-remarriage-fertility-japan 
This repository contains replication files for the following article:

> Raymo, James M; Yoda, Shohei; Iwasawa, Miho; Fukuda, Setsuya. (2026).  
> **Divorce, Remarriage, and Fertility in Japan: A Research Note**  
> *Demography*.

## Data Availability
The analyses use microdata from the **Japanese National Fertility Surveys (JNFS)** conducted by the [National Institute of Population and Social Security Research (IPSS)](https://www.ipss.go.jp/index.asp).  
Access to these microdata is restricted under the Japanese Statistics Act, and therefore the original microdata cannot be shared publicly in this repository.

Permission to use the data was obtained through a research project of the Department of Population Dynamics Research, IPSS, in accordance with Article 32 of the Statistics Act.  
Researchers may apply for secondary use of the JNFS microdata under Article 33 of the Statistics Act of Japan.

### 00_Data
This folder contains **aggregate data files** in tidy-data format that are used to generate the figures reported in the manuscript. These aggregate datasets were constructed from the JNFS microdata.

- `tbl_exp_NofBth_c.rds`: tibble used to generate the results for **cohort** fertility measures
- `tbl_exp_NofBth_p.rds`: tibble used to generate the results for **period** fertility measures

### 01_Script
This folder contains the `R` scripts used to generate the results.

- `00_SetParameters.R`  
  - Loads required libraries and sets global parameters.

- `01_MakeTidyData.R`  
  - Selects and recodes variables, filters analytic samples, and constructs the tibbles used for tables and figures.  
  - *Note:*
    - Running this script requires access to the JNFS microdata. The resulting tibbles are already saved in `00_Data`.
    - The JNFS microdata used in this study were harmonized across survey waves by the authors. As a result, this script cannot be applied directly to the raw JNFS microdata obtained through a secondary-use application. Questions regarding the data harmonization procedures can be directed to the authors.

- `02_Analysis_cohort.R`  
  - Produces results for cohort fertility measures (i.e., cohort completed fertility: CCF, and parity progression ratio: PPR)

- `02_Analysis_period.R`  
  - Produces results for cohort fertility measures (i.e., period total fertility rate: PTFR, and parity progression ratio: PPR)

### 02_Output
This folder stores figures generated by the `R` scripts above

## How to Replicate

You can replicate the main findings and figures using the provided aggregated data.

### 1. Reproducing the Analyses and Figures
1. Open the `R` Project file (`divorce-remarriage-fertility-japan.Rproj`) to set your working directory.
2. Run `01_Script/02_Analysis_cohort.R` to perform the cohort-based and produce the main figures.
3. Run `01_Script/02_Analysis_period.R` to perform the period-based analysis.

All generated plots will be exported as PDF files into the `02_Output/Cohort/` and `02_Output/Period/` directories.

### 2. (Optional) Processing Raw Microdata from Scratch
If you have obtained the official JNFS microdata (`.dta` files) and wish to process them from scratch:
1. Open `01_Script/00_SetParameters.R` and modify the `JnfsDir` variable to point to your local folder containing the JNFS `.dta` files.
2. Run `01_Script/01_MakeTidyData.R` to construct the person-month life history data. This will recreate and overwrite the `.rds` files in the `00_Data/` folder.
